扩展介绍

这是一个由"一只数学柴"开发的Scratch扩展模板，展示了如何创建自定义Scratch扩展。该模板包含了各种类型的积木块示例，可以帮助开发者快速上手扩展开发。

功能特性

  • 多种积木类型示例（命令块、报告块、布尔块等）
  • 带参数的积木示例
  • 菜单选项示例
  • 按钮交互示例
  • 标签分类示例
  • 多语言支持

积木说明

标签和按钮

  • 标签：用于在积木区域进行分类标识
  • 按钮：可点击按钮，点击后会弹出提示框

命令块

  • 命令块：执行基本操作，会弹出提示框显示"Hello, A math stick figure!"
  • 带单个参数的命令块：接受一个文本参数并显示在提示框中
  • 带多个参数的命令块：接受三个文本参数并连接显示在提示框中
  • 带菜单的命令块：提供预设选项菜单供选择
  • 带可插入命令块的菜单命令块：菜单支持插入其他报告块
  • 不能在下方连接积木的命令块：终端命令块，无法在其下方连接其他积木

报告块

  • 报告块：返回文本"Hello, A math stick figure!"，可在舞台查看数值
  • 不能在舞台查看的报告块：同样返回文本，但不能在舞台监视器中查看

布尔块

  • 布尔块：返回真值(true)，可在舞台查看布尔值状态

使用方法

  1. 将扩展文件加载到Scratch环境中
  2. 在扩展面板中找到"一只数学柴的扩展模板"
  3. 点击添加所需的积木到项目中
  4. 根据需要连接和使用积木

Extension Introduction

This is a Scratch extension template developed by "A Math Stick Figure" that demonstrates how to create custom Scratch extensions. This template includes examples of various types of blocks to help developers get started with extension development quickly.

Features

  • Examples of multiple block types (command blocks, reporter blocks, boolean blocks, etc.)
  • Examples of blocks with parameters
  • Menu options examples
  • Button interaction examples
  • Label categorization examples
  • Multilingual support

Block Descriptions

Labels and Buttons

  • Label: Used for category identification in the block area
  • Button: Clickable button that shows an alert when clicked

Command Blocks

  • Command block: Executes basic operations and shows an alert with "Hello, A math stick figure!"
  • Command block with single argument: Accepts a text argument and displays it in an alert
  • Command block with multiple arguments: Accepts three text arguments and concatenates them in an alert
  • Command block with menu: Provides preset menu options to choose from
  • Command block with insertable menu: Menu supports inserting other reporter blocks
  • Terminal command block: Cannot connect other blocks below it

Reporter Blocks

  • Reporter block: Returns text "Hello, A math stick figure!" and can be monitored on stage
  • Non-monitorable reporter block: Also returns text but cannot be viewed in stage monitors

Boolean Blocks

  • Boolean block: Returns true value and can be monitored as a boolean on stage

Usage

  1. Load the extension file into the Scratch environment
  2. Find "A Math Stick Figure's Extension Template" in the extension panel
  3. Click to add the desired blocks to your project
  4. Connect and use the blocks as needed
